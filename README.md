Proyek ini merupakan dashboard interaktif menggunakan Power BI untuk menganalisis performa pemain sepak bola berdasarkan rating, nilai pasar, gaji, distribusi posisi, dan struktur umur tim.

- Menilai struktur umur rata-rata skuad
- Mengukur kualitas kedalaman skuad (substitusi)

## ğŸ“Š Insight Utama (Contoh: FC Barcelona 2019)

- Lini serang memiliki rata-rata rating tertinggi (90+), menunjukkan dominasi ofensif.
